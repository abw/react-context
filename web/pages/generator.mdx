# Generator

The `Generator` function takes one or two arguments.

```js
// Single Model argument
Generator(Model)

// Model and options
Generator(
  Model,
  {
    defaultState: { ... },
    context: MyContext
  }
)
```

## Model

The first mandatory argument is the context model.  This can be a functional
React component or a class-based component.

It will be passed an additional `render` property.  To render the child content
the model should call this function passing any properties that it wants to
make available to consumers.

## Options

The second optional argument is an object containing configuration options.

### `defaultState`

An object containing the default state for the context.  If unspecified it
defaults to an empty object `{ }`.

### `context`

An existing React context object.  If unspecified it create a new context
object for you.
